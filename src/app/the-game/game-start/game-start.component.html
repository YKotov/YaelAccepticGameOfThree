Welcome into my implementation of "The Game of Three"

Below you could see simple rules for this game:

  • The Game starts after inserting your number or generation of random number
  • The receiving player must now add one of -1, 0 or 1 to get a number that is divisible by 3 and then divide it
  • The same rules apply until one of the players reaches the number 1 after division, which ends the game
  • The player reached the 1 is the winner of the game.

Good luck!
